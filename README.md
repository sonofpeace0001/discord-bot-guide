# Discord Bot Build & Deploy Guide

A premium black-and-white landing page that walks readers through building a Discord bot with **CREAO**, pushing it to **GitHub**, and deploying it live on **Railway** — including a copy-paste CREAO prompt template that generates a production-ready discord.js v14 bot.

## What's inside

- `index.html` — the full landing page (self-contained, inline CSS/JS, responsive)
- `vercel.json` — clean routing config for Vercel
- `.gitignore` — ignores common local noise

The page contains:
- 6-step walkthrough (CREAO sign up → prompt → Discord app → GitHub → Railway → verify)
- A copyable CREAO prompt template for building a discord.js bot
- FAQ section
- Affiliate sign-up CTAs throughout

## Deploy on Vercel

1. Go to [vercel.com/new](https://vercel.com/new) and sign in with GitHub.
2. Import this repository (`sonofpeace0001/discord-bot-guide`).
3. Vercel auto-detects the static site — no build settings needed.
4. Click **Deploy**. Your page goes live at `https://discord-bot-guide.vercel.app` (or your custom domain).

That's it. Push any update to `main` and Vercel redeploys automatically.

## Local preview

Just open `index.html` in a browser. No build step.
